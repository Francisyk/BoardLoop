## What Is BoardLoop?
### Boardloop is a **fictional** project that reimagines the way customers interact with board games in stores. The platform allows users to select a board game, rent it, play, and return it effortlessly, following the slogan: ***“rent. play. return. repeat.”***. The design draws inspiration from RPG aesthetics, creating a visually engaging and thematic experience for the user.

![tumblr_7146a3c51090e8bbaef71059cdf8c228_6f105e0f_500](https://github.com/user-attachments/assets/6c85951c-eeed-4f07-ae6f-8987b426141f)

## Technologies Used:
### This project was developed using the following web technologies:
#### **HTML:** The backbone of the structure, ensuring smooth navigation and accessibility.
#### **CSS:** Styling that brings the RPG-inspired theme to life with immersive colors and layouts.
#### **JavaScript:** Adds interactivity, making the rental process dynamic and user-friendly.
